## Webpack - Handlebars template

- built in `handlebars (.hbs)` templating
- `scss` support
- `dev-server` configuration
- support for multiple `html` files
- built in `js/css/images` optimizations

### Commands:
---
Install: 
```
npm install
```
Development:
```
npm run dev
```

Production: 
```
npm run build
```
